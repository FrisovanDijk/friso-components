Colors

About these colors

Many web applications use standard patterns for color palettes. They have a primary and secondary brand color. This template has space for multiple accent colors.

All these styles are computed in _colors.scss. You can find the input values in _variables.scss. This page gives an overview of the possibilities.

Using colors

All background colors displayed here are part of a single map.

Color palette:
    map-get(map-get($colors, $color), $tone)        // Get a color
    color($color[, $tone = base])                   // Shorthand function
    color(secondary, darken)                        // #c8d168

Text color:
    map-get($text-colors, $tone)                    // Get a text color
    text-color($tone: 'copy')                       // Shorthand function
    text-color(faded)                               // #dbe6ec

Accessible contrast text/background:
    adaptive-text-color($background)                // WCAG compliant text contrast function
    adaptive-text-color(color(secondary, darken))   // #fffffe

Default classes

Base colors don't require a [tone] modifier.
    bg-gray-base becomes bg-gray

background color:       bg-[color]-[tone]
text-color:             c-[color]-[tone]
wcag text & background: bg-wcag-[color]-[tone]
hover bg:               hover-bg-[color]-[tone]
hover text:             hover-[color]-[tone]
highlight bg:           hgl-[color]             only base colors
