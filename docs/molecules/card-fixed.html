Fixed cards

 

Full options for vars:

url         (optional)
target      (optional) url target attribute
title       (recommended)
description (optional)
img_url     (optional)
img_alt     (optional)
cover?      (optional) default false
flex?       (optional) default false (not guaranteed to work)
wide?       (optional) default false
short?      (optional) default false
class       (optional)

wide? and short? override the fixed? parameter.
