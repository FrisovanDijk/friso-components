Ghost menu

A lightweight ghost menu

IMPORTANT Place in a flexbox to have it automatically size and center, otherwise it will fill container width.

Full options and html:

name     ghost-menu
vars
    color       (optional) default primary
    items [
        name    (recommended)
        url     (optional)
    ]
    border?     (optional) default false

<nav class="w-80 flex center pt3 f6 ttu">
    <ul class="list flex flex-wrap justify-center items-center center ph2 pv1
    {{#border?}}
        br1 b-{{color}} ba
    {{/border?}}">
    {{#items}}
    <li><a class="db link c-{{color}}
    pa2 no-underline dim" href="{{url}}" target="{{target}}">{{name}}</a></li>
        {{/items}}
    </ul>
</nav>
