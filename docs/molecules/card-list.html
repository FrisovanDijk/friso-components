List cards

List cards are used to generate a list on a card element.

Full options for data:

title                   (recommended)
description             (optional)
items          name     (recommended)
               url      (optional)
               img_url  (optional) TODO
               img_alt  (optional) TODO
striped?                (optional) default false
border?                 (optional) default false
