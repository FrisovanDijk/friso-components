Tag list

A list of tags with light background color and optional links. Only works with brand colors.

Full options and html:

name     tag-list
vars
    title       (optional)
    color       (optional) light4 type of specified color
                , default secondary-light4
    tags[
        name    (recommended)
        url     (optional)
    ]

<div class="flex items-center">
    <div class="ma1 pv1 ph2">{{title}}</div>
    <div class="flex flex-wrap">
        < class="link" href="{{url}}">
            <div class="ma1 pv1 ph2 br2
            bg-wcag-{{color}}-light4">
                {{name}}
            </div>
        </a>
    </div>
</div>
