Breadcrumbs

Full options and template:

name     breadcrumbs
vars
    items [
        name     (recommended) Name of breadcrumb.
        url      (recommended) URL for name
        current? (optional)    Current position in breadcrumb trail
    ]

<div class="w-100 ma1 pv1 pl1 br1">
    {{#items}}
        {{#url}}
            <a class="f6 c-copy no-underline hover-underline
            {{#current?}}b{{/current?}}" href="{{url}}">
        {{/url}}
        / {{name}}
        {{#url}}</a>{{/url}}
    {{/items}}
<div>
